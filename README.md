# Home AI — Smart Home Companion

> *Your house remembers. A cocapn vessel for the connected home.*

## What It Is

A repo-native AI companion for smart home management. It learns your routines, remembers your preferences, and grows smarter over time. Not a voice assistant that forgets — a persistent agent that accumulates context about your home and your life.

## Features

- 🏠 **Routine learning** — learns your daily patterns and anticipates needs
- 💡 **Device coordination** — manages lights, thermostat, security across systems
- 📊 **Energy tracking** — monitors usage patterns and suggests optimizations
- 🔒 **Privacy-first** — all data in your repo, no cloud dependency
- 🔄 **A2A ready** — talks to other vessels in your fleet

## Deploy

```bash
git clone ./.git
cd home-ai
npm install
npx wrangler dev
```

## Status

✅ Boots locally. Core chat and device management working.
